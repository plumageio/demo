# Plumage demo

> Prototyping the plumage flow 
> https://www.plumage.io/demo/

Simple HTML prototype for user testing the plumage flow.

### Deployment

```
./deploy.sh
```

Requires the following environment variables to be set before executing:

```bash
DEPLOY_OWNER
DEPLOY_OWNERGROUP
DEPLOY_SRC
DEPLOY_USER
DEPLOY_SERVER
DEPLOY_PATH
```
